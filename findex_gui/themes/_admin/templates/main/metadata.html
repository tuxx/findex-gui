Add database

By default, findex doesn't come with metadata. In general; the metadata includes:

  • Movie posters, ratings, actors/directors
  • Series posters, ratings, actors
  • Music/Band information

Findex can work without this metadata but for user-experience it's better that it is present.

The dump itself is provided as a zip file, with the following naming convention: findex_meta_*.zip, where the * is a number. This signifies the metadata database version number.

The dump is available at a webpage: blog.cedsys.nl/findex_meta.html. Fetch the file and upload it here, to start the bootstrapping process.

Please note that:

  • The upload can take up to 2 minutes to import. Just let it load.
  • The program unzip most be present on the current system
  • I try to update the webpage as much as I can, so look regularly for new dumps.
