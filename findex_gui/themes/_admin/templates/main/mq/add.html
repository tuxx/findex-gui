Add message queue

  1. Message Queue
  2. Add

Info

This page lets you add a "message broker" to Findex. One such broker is RabbitMQ.

The following broker types are currently supported:

  • RabbitMQ

After adding the broker, you may assign it to a (resource) group.

API

URL: /api/v2/mq/add

Authentication via HTTP BASIC (or cookie) required.

Python

import requests

data = {
    "name": "My AMQP server" % i,
    "broker": "rabbitmq",
    "host": "127.0.0.1",
    "vhost": "/",
    "port": 5672,
    "queue": "my_queue",
    "ssl": False,
    "auth_user": "guest",
    "auth_pass": "guest"
}

auth = ('root', 'changeme')

r = requests.post('http://localhost:9200/api/v2/mq/add',
                  json=data, auth=auth)
