Findex is an open-source search engine for files.
Findex can gather files from FTP, HTTP and SMB resources all over the world Findex is fast and runs on (almost) any machine capable of running Linux. Use the REST API to implement your own file searching tool Clone/Fork Findex on GitHub. Contributions are more than welcome!

Built on:

Findex is a high performance platform for finding and indexing files. It tries to be fast, customizable and scaleable. Findex is a multi-purpose search engine for alternative protocols.

Installation Guide

Hosting your own instance is easy. All you need is a Linux box and a Postgres database.

Distributed Crawling

Findex supports distributed crawling out of the box.
